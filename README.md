# Bhupendra Singh High School — Static Site

यह एक प्रारम्भिक स्टैटिक वेबसाइट है जो Bootstrap का उपयोग करती है।

To preview locally: open any html file in a browser.

To deploy on GitHub Pages: enable Pages from repository settings and set branch to `main` (or the branch you push to).

Customize content by editing the HTML files and replace placeholder images in `assets/img/`.
